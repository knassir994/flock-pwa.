# قطيعي — الطريق لـ Google Play Store

هاد الملفات صاوبينها باش يمشيو للطريق ديال Play Store. هادشي غير معناه أنا مانقدرش نديرو نيابة عنك (خاصو حساب مطور ديالك)، ولكن هاد الملفات هي كامل الجانب التقني لي خاص.

## المرحلة 1: استضافة الموقع (مجانا)
لازم الملفات يكونو مستضافين على رابط حقيقي (https). أسهل طريقة مجانية:
1. صاوب حساب فـ GitHub (github.com)
2. صاوب "repository" جديد، رفع ليه هاد الملفات كاملين (index.html, manifest.json, service-worker.js, icon-192.png, icon-512.png, icon-512-maskable.png)
3. فـ Settings > Pages، فعّل GitHub Pages
4. غادي يعطيك رابط بحال: `https://username.github.io/repo-name/`

(بدائل أخرى: Netlify أو Vercel، بزاف منها بسيطين ومجانيين بزاف)

## المرحلة 2: تحويلو لتطبيق أندرويد (.aab)
1. دخل لـ **pwabuilder.com** (مجاني)
2. دخل الرابط ديال الموقع لي صاوبتي
3. اختار "Android" package
4. حمّل ملف الـ `.aab` لي غيعطيك

## المرحلة 3: حساب Google Play Developer
1. دخل لـ **play.google.com/console**
2. خلص 25 دولار (مرة وحدة، ماشي شهرية)
3. أكد الهوية ديالك (كايطلب وثيقة هوية فبعض البلدان)

## المرحلة 4: نشر التطبيق
1. صاوب "تطبيق جديد" فـ Play Console
2. زيد ملف الـ `.aab`
3. عمر معلومات: اسم، وصف، صور من التطبيق (screenshots)، سياسة الخصوصية (لازمة حتى إلا التطبيق بسيط)
4. صيفط للمراجعة — عادة كتاخد من يوم لـ أسبوع باش غوغل يوافق عليه

## ملاحظة مهمة
بدّلت طريقة حفظ البيانات فهاد النسخة (من نظام Claude الخاص لـ `localStorage` العادي) باش تخدم فأي متصفح ولا تطبيق، بلا ما تكون محتاجة Claude. البيانات كتبقى محفوظة فالتيليفون ديال المستخدم فقط.
